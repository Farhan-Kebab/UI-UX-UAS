- Format: JPG atau PNG
- Ukuran rekomendasi: 400x300px
- Ukuran file: < 100KB per file untuk performa optimal

## Cara Penempatan:
Letakkan semua gambar produk di folder ini: `dapur-roti/assets/img/products/`

Jika Anda ingin menambahkan gambar, cukup copy file gambar ke folder ini dan pastikan nama file sesuai dengan yang terdaftar di `js/produk.js`.
